# Генератор цветового фона
React-приложение для генерации случайных цветов с каждым кликом.

## Основные функции
- Генерация случайного цвета фона при каждом клике
- Отображение цвета в HEX-формате
- Плавные анимации переходов
- 
## Процесс использования
Запуск приложения:
- При открытии приложения автоматически генерируется случайный цвет фона
- Цвет отображается в виде фона всей страницы
- HEX-значение цвета показывается в центральном блоке

Смена цвета:
- Для получения нового цвета нужно нажать на кнопку "Сгенерировать цвет"
- При каждом клике создаётся совершенно новый случайный цвет

Отображение информации:
- В центре экрана отображается текущий цвет в формате HEX (например: #A3D5F7)
- Цветовой превью-квадрат показывает текущий цвет в уменьшенном виде
- При наведении на превью-квадрат он немного увеличивается
